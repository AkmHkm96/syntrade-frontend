## Run It Locally

Clone repository

```
git clone git@github.com:ariessa-deriv/syntrade-frontend.git
```

Create .env file and insert values

```
NEXT_PUBLIC_V3_SITE_KEY=
RECAPTCHA_V3_SECRET_KEY=
NEXT_PUBLIC_API_URL=
```

Install packages using npm

```
npm install
```

Start Next.js app in development

```
npm run dev
```
